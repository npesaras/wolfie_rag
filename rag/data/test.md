# Test Document

This is a test document for the file watching functionality.

## Features

- Automatic file detection
- Document processing with Docling
- Vector embeddings with Ollama
- RAG queries

## How it works

When you add files to the `data/` directory, they are automatically:
1. Detected by the file watcher
2. Processed by Docling
3. Chunked and embedded
4. Stored in the vector database
